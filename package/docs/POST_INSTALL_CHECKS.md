# Post-Install Checks

1. Enable `marketplace` for a test role.
2. Open `/dashboard/marketplace` and confirm the catalog renders.
3. Search for a module and confirm the result count updates.
4. Select each category and confirm only matching modules remain.
5. Confirm trust, compatibility, publisher, and lifecycle status are visible.
6. Confirm `Review package` enters metadata inspection only and does not execute package code.
7. Disable `marketplace` and confirm its navigation item is hidden.
8. Confirm disabling the module does not remove registered module metadata.
